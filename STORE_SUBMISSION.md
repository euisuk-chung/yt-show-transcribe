# Chrome Web Store 등록 가이드

## 준비 사항

### 1. 필수 파일 준비

- [x] manifest.json
- [x] 소스 코드 파일들
- [ ] 아이콘 이미지 (3가지 크기)
- [ ] 스크린샷 (최소 1개, 권장 3-5개)
- [ ] 프로모션 이미지 (선택사항)

### 2. 아이콘 요구사항

Chrome Web Store에 필요한 아이콘:

- **128x128 픽셀** (필수) - 스토어 목록에 표시
- **48x48 픽셀** (권장) - 확장 프로그램 관리 페이지
- **16x16 픽셀** (권장) - 파비콘

### 3. 스크린샷 요구사항

- **크기**: 1280x800 또는 640x400 픽셀
- **형식**: PNG 또는 JPEG
- **개수**: 최소 1개, 최대 5개 권장
- **내용**: 확장 프로그램의 주요 기능을 보여주는 화면

## 등록 프로세스

### 1단계: Chrome Web Store 개발자 계정 등록

1. [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole) 접속
2. Google 계정으로 로그인
3. **일회성 등록 수수료 $5 결제** (신용카드 필요)
4. 개발자 계정 정보 입력

### 2단계: 확장 프로그램 패키징

확장 프로그램을 ZIP 파일로 압축:

```bash
# Windows (PowerShell)
Compress-Archive -Path * -DestinationPath yt-transcript-auto-show.zip

# 또는 수동으로
# 프로젝트 폴더의 모든 파일을 선택하여 ZIP으로 압축
# (폴더 자체가 아닌 파일들을 직접 선택)
```

**주의**: ZIP 파일 내부에 프로젝트 폴더가 아닌 파일들이 직접 있어야 합니다.

올바른 구조:
```
yt-transcript-auto-show.zip
├── manifest.json
├── background.js
├── content.js
├── popup.html
├── popup.js
├── popup.css
└── icons/
```

잘못된 구조:
```
yt-transcript-auto-show.zip
└── yt-show-transcribe/  ❌ (폴더가 한 번 더 감싸면 안됨)
    ├── manifest.json
    └── ...
```

### 3단계: 스토어에 업로드

1. [Developer Dashboard](https://chrome.google.com/webstore/devconsole) 접속
2. "새 항목" 버튼 클릭
3. ZIP 파일 업로드
4. 자동 검증 통과 확인

### 4단계: 스토어 정보 입력

#### 필수 정보

**제품 세부정보**:
- **이름**: YouTube Transcript Auto Show
- **요약** (132자 이내):
  ```
  유튜브 영상의 스크립트를 자동으로 표시합니다. 토글 버튼으로 쉽게 켜고 끌 수 있습니다.
  ```

**세부 설명**:
```
유튜브 영상의 스크립트(대본)를 매번 수동으로 켜는 것이 불편하셨나요?

이 확장 프로그램은 유튜브 영상 페이지에 접속하면 자동으로 스크립트를 표시해줍니다.

주요 기능:
✅ 유튜브 영상 페이지 접속 시 자동으로 스크립트 표시
✅ 토글 버튼으로 쉽게 켜고 끄기 가능
✅ 설정이 자동으로 저장됨
✅ 페이지 이동 시에도 자동 적용

사용 방법:
1. 확장 프로그램 아이콘 클릭
2. 토글 스위치로 기능 켜기/끄기
3. 유튜브 영상 페이지 방문 시 자동으로 스크립트 표시

개인정보 보호:
- 어떠한 데이터도 수집하지 않습니다
- 유튜브 페이지에서만 작동합니다
- 오픈 소스 (GitHub에서 소스 코드 확인 가능)

문의사항이나 버그 리포트는 GitHub 저장소에서 이슈로 등록해주세요.
```

**카테고리**: 생산성(Productivity) 또는 도구(Utilities)

**언어**: 한국어 (Korean)

#### 그래픽 에셋

1. **아이콘** (128x128): 업로드
2. **스크린샷**: 최소 1개 업로드
   - 추천 스크린샷:
     - 유튜브 페이지에서 스크립트가 표시된 화면
     - 확장 프로그램 팝업 화면
     - 토글 스위치 사용 예시

3. **프로모션 타일** (440x280, 선택사항)
4. **마케팅 타일** (1400x560, 선택사항)

#### 개인정보처리방침

간단한 확장 프로그램이므로 다음과 같이 작성 가능:

```
이 확장 프로그램은 어떠한 개인정보도 수집하지 않습니다.

- 사용자 데이터를 수집하지 않습니다
- 외부 서버로 데이터를 전송하지 않습니다
- Chrome Storage API를 사용하여 사용자의 설정만 로컬에 저장합니다
- 유튜브 페이지에서만 작동합니다

소스 코드는 GitHub에서 공개되어 있습니다: [GitHub 저장소 URL]
```

또는 GitHub Pages나 블로그에 개인정보처리방침 페이지를 만들고 URL을 제공할 수 있습니다.

### 5단계: 배포 설정

**공개 범위**:
- **공개**: 모든 사용자에게 공개
- **비공개**: 특정 이메일 주소에만 공개
- **테스터**: 테스터 그룹에만 공개 (베타 테스트용)

**지역**: 전 세계 또는 특정 국가 선택

**가격**: 무료

### 6단계: 제출 및 검토

1. "검토를 위해 제출" 버튼 클릭
2. Google의 자동 및 수동 검토 대기 (보통 1-3일)
3. 승인 시 자동으로 스토어에 게시됨

## 검토 체크리스트

승인 확률을 높이기 위한 체크리스트:

- [ ] manifest.json의 모든 권한이 명확하게 설명됨
- [ ] 개인정보처리방침이 명시됨
- [ ] 스크린샷이 실제 기능을 보여줌
- [ ] 설명이 명확하고 정확함
- [ ] 악성 코드나 난독화된 코드가 없음
- [ ] 모든 외부 라이브러리가 명시됨 (해당사항 없음)

## 업데이트 프로세스

확장 프로그램 업데이트 시:

1. manifest.json의 version 번호 증가
   ```json
   "version": "1.0.1"  // 1.0.0 -> 1.0.1
   ```

2. 새로운 ZIP 파일 생성
3. Developer Dashboard에서 "패키지 업로드" → "새 패키지 업로드"
4. 변경 사항 설명 작성
5. "검토를 위해 제출"

## 자주 묻는 질문

**Q: 검토에 얼마나 걸리나요?**
A: 보통 1-3일 정도 걸립니다. 복잡한 확장 프로그램은 더 오래 걸릴 수 있습니다.

**Q: 거부되면 어떻게 하나요?**
A: 거부 사유를 확인하고 수정 후 다시 제출할 수 있습니다.

**Q: 개인정보처리방침이 꼭 필요한가요?**
A: 네, 2021년부터 모든 확장 프로그램에 필수입니다.

**Q: 수수료 $5는 환불되나요?**
A: 아니요, 개발자 등록 수수료는 환불되지 않습니다.

**Q: 여러 확장 프로그램을 등록할 수 있나요?**
A: 네, 한 번 $5를 결제하면 무제한으로 등록 가능합니다.
